Project Description

Sales Dashboard that was built aims to display Client transaction data in a user-friendly manner.

In this project there are several worksheets, including the sales summary worksheet which contains a summary of sales to expenses which is expressed predominantly in the form of charts,
sales by customer worksheets which function to monitor transactions per customer so you can see which customers are classified as Royal or passive,
sales by product worksheet which aims to display the largest or smallest sales based on product,
and the Grossprofit Worksheet where this worksheet functions to display the grossprofit value, the comparison of revenue with Grossprofit and the number of transactions. All these worksheets cannot be separated from filtering based on period, salesman, brand and so on
