Project Description

Profit and Loss Dashboard that was built aims to display Client transaction data in a user-friendly manner.

In this project there are several worksheets containing dashboard displays, including:
Profit and Loss worksheets which function to display the value of each Chart of Account,
Leadger Overview Worksheets which function to view transaction movements in each Chart of Accounts based on period,
Balance Sheet worksheet that will display the values ​​of Liabilities, Assets, and Equity in a company, and so on
